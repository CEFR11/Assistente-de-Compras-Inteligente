# Assistente-de-Compras-Inteligente
O Assistente de Compras Inteligente interpreta buscas escritas em linguagem natural e retorna recomendações de produtos de forma organizada. O usuário pode escrever frases como "celular barato com boa câmera" e o sistema identifica categoria, faixa de preço e atributos, realizando a busca nas estruturas de dados adequadas.
